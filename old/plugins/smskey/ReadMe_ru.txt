 Этот плагин позволяет обеспечить платный доступ к чему-либо на вашем сайте. 
 В ответ на присланное смс-сообщение пользователю приходит короткий текстовый пароль (ключ),
 с помощью которого он может получить доступ к тому, что вы закроете этим ключом. 
 Вы сами решаете сколько раз или как долго можно использовать пароль для доступа.

Модуль оплаты скрытого текста посредством отправки смс-сообщения.

Для работы модуля требуется регистрация на сайте :smscoin.com

Использование: [sms] Скрытый текст [/sms] 


Инстaляция:
===============================================================================
Файл: smskey.php					Редактировать-> Сохранить
===============================================================================
1. Найдите блок:
	########### SMSCoin key ##########	
		$key_id = 111111;	//ИД смс:ключа
		$s_enc = 'UTF-8';	//Кодиривка скрипта windows-1251,UTF-8,..
		$tag_name = 'sms';	//Имя тага 
		$s_lang = 'english';     //Язык интерфейса по умолчанию
		// Поддерживаемые языки:
		//{"russian", "belarusian", "english", "estonian", "french", "german", "hebrew", "latvian", "lithuanian", "romanian", "spanish", "ukrainian"}
	##################################

заполните требуемые значения и сохраните.


2. Скопируйте папку "smskey" в папку  ./e107_plugins/ .

===============================================================================		
В Админ панеле :

 Plugins -> Plugin Manager ->Install
